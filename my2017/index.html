2017年的小目标

新的一年即将到来，给我的2017定一些小(chui)目(niu)标(bi)。

小目标制定

1月份目标

  • 精读《深入理解Java虚拟机》并输出读书笔记；
  • 泛读The Java Virtual Machine Specification;
  • 阅读Spring Framework Reference前三部分（概述、版本新特性、核心技术）并输出笔记；
  • 泛读《RabbitMQ实战》；
  • 泛读《Java网络编程》；
  • 了解微信小程序，阅读相关的官方介绍和文档；
  • 阅读InfoQ全年经典博文；
  • 阅读阿里巴巴双11电子书;
  • 阅读Elastic{ON}开发者大会PPT;
